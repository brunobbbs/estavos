Como as turmas são divididas?

Na unidade Plano Piloto (ASCADE), cada curso do APRENDA é dividido em duas turmas. Uma para adultos e outra para crianças.

Dessa forma nosso instrutor pode utilizar didáticas diferentes para que nossos alunos possam ter o máximo de aproveitamento.

Na unidade de Águas Claras só há turma para crianças até o momento.

Sobre Xadrez

O que é Xadrez?

Xadrez é uma emocionante batalha com exércitos de cavaleiros, torres, arqueiros e corajosos guerreiros compartilhando um mesmo objetivo, aprisionar o rei adversário e vencer a batalha.

O Xadrez ajuda nos estudos?

Sim! Pesquisas apontam que estudantes que jogam xadrez têm uma melhoria de 15% em provas de matemática após o início da prática.

Quais habilidades são trabalhadas com Xadrez?

O Xadrez trabalha naquele que o pratica, uma série de habilidades cognitivas tais como atenção, concentração, raciocínio lógico, memória, organização de ideias, antecipação, espírito de decisão, autocontrole, disciplina, perseverança, dentre outros.

A partir de qual idade posso aprender?

O Xadrez é recomendado para todas as faixas etárias. Crianças podem tirar proveito do jogo a partir de 6 anos de idade.

Não tenho paciência, Xadrez não é para mim?

Essa é uma das maiores falácias a respeito deste nobre jogo-esporte-ciência. Não é preciso ter paciência para jogar Xadrez, este é um dos atributos que você adquire com a prática do esporte.

Tenho TDAH, posso aprender Xadrez?

Sim. Pesquisas apontam que o jogo de xadrez é um excelente aliado no tratamento do Transtorno de Déficit de Atenção e Hiperatividade (TDAH). Maior concentração em sala de aula e desenvolvimento mais rápido do raciocínio são alguns dos benefícios de quem joga.
